# 📖 Digital Life Lessons - Client 

**Live Website:** [https://digitallife-client.vercel.app/](https://digitallife-client.vercel.app/)

## 🎯 Project Purpose
Digital Life Lessons is a community-driven platform where users can create, store, and share meaningful life lessons, personal growth insights, and wisdom they have gathered over time. The platform helps preserve personal wisdom, encourages mindful reflection, and allows users to grow by exploring lessons from the community.

## ✨ Key Features
* **Authentication & Roles:** Secure login via Email/Password and Google using Better Auth. Features distinct User and Admin roles.
* **Premium Memberships:** Users can upgrade to a Premium plan via a one-time Stripe checkout to unlock exclusive lessons.
* **Dynamic Dashboard:** Personalized user dashboards to manage created lessons, view saved favorites, and track engagement stats.
* **Admin Command Center:** Comprehensive admin tools to manage users, monitor platform revenue, and moderate reported content.
* **Community Interaction:** Users can like, comment on, and save lessons to a personalized favorites list.
* **Advanced Filtering:** Search public lessons by title, and filter by category or emotional tone.
* **Smooth Animations:** Engaging UI interactions powered by Framer Motion.

## 📦 NPM Packages Used
* `react` / `next` (Core Framework)
* `axios` (Data fetching and API requests)
* `better-auth` (Authentication handling) 
* `framer-motion` (Scroll and layout animations) 
* `react-hot-toast` (Sleek success/error notifications)
* `react-share` (Social media sharing functionality) 
* `lucide-react` / `react-icons` (UI Iconography)
* `tailwindcss` (Utility-first styling)
